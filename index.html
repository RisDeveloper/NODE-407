Tech Stack

Teknologi yang digunakan untuk membangun NODE-407

PHP 8.2

Backend API dan logic aplikasi menggunakan PHP 8.2 dengan arsitektur PDO untuk koneksi database.

PostgreSQL + Supabase

Database relasional di-hosting via Supabase dengan koneksi langsung & pooler sebagai fallback.

HTML + CSS / JS

Frontend modern dengan CSS variables, tema Albanian flag (Merah & Hitam), Plus Jakarta Sans font.

Multi AI Models

Integrasi 4 model AI: Llama 3.1, OpenAI GPT, Google Gemini Flash & DeepSeek.

Docker + Nginx

Deployment containerized dengan Docker (php:8.2-fpm-alpine) dan Nginx sebagai reverse proxy.

Auth & Session

Sistem autentikasi email/password + Google OAuth, session management via PHP sessions dengan audit log.
